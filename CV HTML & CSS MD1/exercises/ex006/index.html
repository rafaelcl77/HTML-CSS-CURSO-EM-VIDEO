Principais formatações

Negrito / Destaque

Vou criar um negrito não semantico usando a tag B

Criando um paragraph em negrito com a teg strong que é a forma semantica

Italico / Ênfase

Agora temos um termo em Italico usando a tag I(não semantica)

Esta frase agora esta em enfase(semantico) usando a tag EM

Texto Marcado - marcador

Podemos tambem criar uma texto marcado usando a tag MARK

um texto bem pequeno usando a tag small

Texto Deletado

Podemos marcar um texto como excluido para indicar que dever ser lido,mas não considerado usando a tag DEL

Texto Enserido

Podemos marcar um texto inserido para dar uma enfase e indicar que ele foi acionado depois usando a tag INS

Texto Sobrescrito

Para enserir coisas do tipo x20+3 -- usando a tag SUP (encima)

Texto Subscrito

Para inserir coisas do tipo H2O -- usando a tag SUB (embaixo)
